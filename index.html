JAVAMySQL

Administrative system

School assignment. An administrative system for Hogwarts that both teachers and students can use. Based on a requirement specification, the system was to be programmed in Netbeans, together with a already existing database. Teachers can add, change or delete students, courses and grades. Teachers can also add or retract points in the House Cup. Students can use the system for information searches, like their grades or courses. I put a lot of effort in making the system user-friendly, and managed to create a system that's easy to navigate and has a nice beginner level of design to it.

HTMLCSSJS

Portfolio website

School assignment. The website were to be a portfolio site with focus on mobile first. It had a requirement specification including a little bit of jQuery, slideshows and a contact form, with realtime validation and regex. The programming was done in Visual Studio Code. The HTML and CSS is validated without errors by the W3C:s markup validation service and is fully responsive. I wanted to push the jQuery aspect of the assignment and made a gallery showing of thumbnails of the photos presented. When clicked a bigger window showing the picture and associated text was shown.

C#

Terminal Dungeon Crawler

Made on my free time. Simplistic super short game created for practising the basics of C# and introducing myself to game design. The player needs to reach the end boss, there is only one way that is correct, and if the player chooses the wrong path they get either a monster fight or treasure. Some features are: leveling up after 2 monsters killed, randomized monster names with random hp & attack damage, chance to find gold, and three different classes to choose from. After the player beats the boss, the stats are saved and listed together with other playthroughs. Programmed in Visual Studio.
